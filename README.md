# Guess the Word Game

A simple word guessing game built with HTML, CSS, and JavaScript.

## How to Play

1. Open `index.html` in your browser
2. A hidden word (Portuguese city name) will be displayed as underscores
3. Click on letter buttons to guess letters
4. Correct letters will be revealed in the word
5. Wrong guesses are tracked (max 9 allowed before game over)
6. Win by guessing all letters before running out of attempts

## Features

- Random word selection from Portuguese cities
- Visual feedback with color-coded wrong attempts counter
- Tracks all wrong letters tried
- Random win/lose messages
- New Game button to restart
